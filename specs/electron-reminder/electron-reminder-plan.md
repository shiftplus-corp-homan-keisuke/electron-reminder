# Electron Reminder - 製品要件定義書 (PRD)

## 概要

Windows向けのデスクトップリマインダーアプリケーション。
シンプルで直感的なUIを持ち、システムトレイに常駐してバックグラウンドでリマインド通知を行う。

## 技術スタック

| カテゴリ | 技術 | 理由 |
|---------|------|------|
| フレームワーク | Electron 35+ | デスクトップアプリ基盤、Windows通知・トレイAPI内蔵 |
| ビルドツール | Electron Forge + Vite | 公式推奨、HMR対応、高速ビルド |
| UI | React 19 + TypeScript | コンポーネントベースUI |
| UIコンポーネント | shadcn/ui | 高品質・カスタマイズ可能なUIライブラリ |
| 状態管理 | Zustand | 軽量、永続化ミドルウェア対応 |
| データ永続化 | IndexedDB (Dexie.js経由) | オフライン対応、大容量、ブラウザ標準API |
| スタイリング | Tailwind CSS 4 | shadcn/uiの依存、ユーティリティファースト |
| 日時処理 | date-fns | 軽量、Tree-shakeable、繰り返し計算に便利 |
| アイコン | Lucide React | shadcn/uiと統合済み |

## 機能要件

### F-01: リマインダーCRUD

| 項目 | 内容 |
|------|------|
| 作成 | タイトル(必須)、メモ(任意)、日時、繰り返し設定を指定して作成 |
| 一覧表示 | 有効なリマインダーを次回実行日時でソート表示 |
| 編集 | 既存リマインダーの全項目を編集可能 |
| 削除 | 確認ダイアログ後に削除 |
| 有効/無効切替 | トグルスイッチでリマインダーを一時的に無効化 |

### F-02: 繰り返し設定

| タイプ | 説明 | 設定項目 |
|--------|------|----------|
| 1回のみ | 指定日時に1回だけ通知 | 日時 |
| 日次 | 毎日同じ時刻に通知 | 時刻 |
| 週次 | 毎週指定曜日の時刻に通知 | 曜日、時刻 |
| 月次 | 毎月指定日の時刻に通知 | 日(1-31)、時刻 |
| 年次 | 毎年指定月日の時刻に通知 | 月、日、時刻 |

**繰り返し計算ルール:**
- 月次で31日指定 → その月の最終日にフォールバック
- 1回のみで過去日時 → 作成時に警告、保存は許可
- 無効化中のリマインダーはスケジュール対象外

### F-03: Windows通知

| 項目 | 内容 |
|------|------|
| 通知方式 | Windows 10/11のトースト通知 (Electron Notification API) |
| 通知内容 | タイトル + メモ(あれば先頭100文字) |
| 通知クリック | アプリウィンドウを表示し、該当リマインダーにフォーカス |
| 通知タイミング | 設定日時の±1分以内に発火 |

### F-04: システムトレイ常駐

| 項目 | 内容 |
|------|------|
| トレイアイコン | アプリ専用アイコン(16x16, 32x32) |
| ウィンドウ閉じ動作 | ウィンドウを非表示(破棄しない)、トレイに残る |
| トレイ左クリック | ウィンドウを表示/非表示トグル |
| トレイ右クリック | コンテキストメニュー表示 |
| コンテキストメニュー | 「開く」「スタートアップ登録」「終了」 |

### F-05: スタートアップ登録

| 項目 | 内容 |
|------|------|
| 登録方式 | `app.setLoginItemSettings()` (Electron内蔵API) |
| UI操作 | トレイメニューまたは設定画面からトグル |
| デフォルト | 未登録(ユーザーが明示的に有効化) |

### F-06: オフライン対応

| 項目 | 内容 |
|------|------|
| データ保存 | IndexedDB (ローカルのみ) |
| ネットワーク依存 | なし(全機能がオフラインで動作) |
| 外部通信 | 一切なし |

## データモデル

### Reminder

```typescript
interface Reminder {
  id: string;                  // UUID v4
  title: string;               // 必須、最大100文字
  memo: string;                // 任意、最大500文字
  dateTime: string;            // ISO 8601 (初回実行日時)
  recurrenceType: RecurrenceType;
  recurrenceConfig: RecurrenceConfig;
  enabled: boolean;            // 有効/無効
  nextFireTime: string | null; // 次回発火予定日時 (ISO 8601)
  createdAt: string;           // ISO 8601
  updatedAt: string;           // ISO 8601
}

type RecurrenceType = 'once' | 'daily' | 'weekly' | 'monthly' | 'yearly';

interface RecurrenceConfig {
  time: string;           // "HH:mm" 形式
  dayOfWeek?: number;     // 0(日)-6(土) - weeklyで使用
  dayOfMonth?: number;    // 1-31 - monthlyで使用
  month?: number;         // 1-12 - yearlyで使用
  day?: number;           // 1-31 - yearlyで使用
}
```

### AppSettings

```typescript
interface AppSettings {
  launchAtStartup: boolean;
  theme: 'light' | 'dark' | 'system';
}
```

## 非機能要件

| 項目 | 基準 |
|------|------|
| 起動速度 | 3秒以内にメイン画面表示 |
| メモリ使用量 | アイドル時150MB以下 |
| スケジュール精度 | 設定時刻±1分以内に通知発火 |
| データ上限 | リマインダー1000件でも操作遅延なし |
| 対応OS | Windows 10 / 11 (64bit) |
| インストーラー | Squirrel.Windows (Electron Forge標準) |

- リマインダーのカテゴリ/タグ

- サウンド通知のカスタマイズ
